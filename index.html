1. Resumen Técnico

Este estudio evalúa el comportamiento del algoritmo de agrupamiento jerárquico aglomerativo (HAC) sobre el dataset Iris (150 observaciones, 4 variables continuas). El análisis demuestra la efectividad de la estandarización multidimensional previa para evitar el sesgo métrico, aplicando el criterio de mínima varianza de Ward. El proceso de fusión se evalúa intrínsecamente mediante el Coeficiente de Silueta y se valida externamente mediante una matriz de contingencia estructural.

2. Formulación Matemática

Fórmula de Lance-Williams

El cálculo recursivo de distancias entre un cluster fusionado $(U \cup V)$ y un cluster externo $W$ se computa mediante:

$$d(U \cup V, W) = \alpha_1 d(U, W) + \alpha_2 d(V, W) + \beta d(U, V) + \gamma |d(U, W) - d(V, W)|$$

Parámetros del Enlace de Ward

Para minimizar la varianza interna de las agrupaciones en cada etapa de fusión, los parámetros de actualización se definen como:

$$\alpha_1 = \frac{N_U + N_W}{N_U + N_V + N_W}$$
$$\alpha_2 = \frac{N_V + N_W}{N_U + N_V + N_W}$$
$$\beta = \frac{-N_W}{N_U + N_V + N_W}, \quad \gamma = 0$$

Estandarización de Variables ($Z$-Score)

Para anular el efecto de la dominancia métrica producida por variables con rangos más amplios (p. ej., longitud del sépalo), cada coordenada es transformada mediante:

$$z_{ij} = \frac{x_{ij} - \mu_j}{\sigma_j}$$

5. Análisis y Complejidad

  • Complejidad Temporal: El cálculo recursivo requiere $\mathcal{O}(N^3)$ en aproximaciones secuenciales elementales, optimizable a $\mathcal{O}(N^2 \log N)$ mediante colas de prioridad.
  • Complejidad Espacial: El algoritmo necesita almacenar una matriz simétrica de disimilitud de tamaño $\mathcal{O}(N^2)$ en memoria dinámica. Esto hace inviable su despliegue en bases de datos con millones de registros activos (Big Data).
